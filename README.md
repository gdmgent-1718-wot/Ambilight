## Deploy

### Clone

```
$ cd ~/Code/
```

```
$ git clone https://github.com/gdm-1718-jefroose/sh_ambilight.git
```

### Install

Install the Sense Hat package on your Raspberry Pi

```
$ sudo apt-get update
```
```
$ sudo apt-get install sense-hat
```
```
$ sudo reboot
```
